Comode\graph\Factory

  • It creates a new node with no value
  • It creates a new node with a string value
  • It creates a new node with a file value
  • It retrieves a node by its id
  • It creates new instances of the same node
  • It retrieves a string value
  • It creates new instances of the same string value
  • It retrieves a file value
  • It creates new instances of the same file value

Comode\graph\Node

  • It provides its own id
  • Its id is an integer
  • It can be connected to other nodes
  • It produces an error when you try to connect it to something else
  • It provides all nodes it is connected to
  • Provided nodes are of the node type
  • It provides its value
  • It does not provide a value when it is empty
  • It copies its value file to an internal location

Comode\graph\Value

  • It provides its node

Comode\syntax\Fact

  • Its answer can be set as a string
  • Its answer can be set as a file

Comode\syntax\Factory

  • It creates statements

Comode\syntax\Statement

  • Facts can be added to it
